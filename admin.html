A file called content.js has been downloaded. Now upload it to your website:

1 Go to your website on GitHub
2 Click on the file named content.js
3 Click the pencil icon (edit button) in the top right
4 Select all text (Ctrl+A) and delete it, then open the downloaded file in Notepad, select all (Ctrl+A), copy (Ctrl+C), and paste (Ctrl+V) into GitHub
5 Click the green "Commit changes" button, then click "Commit changes" again in the popup

Your website will update within a few minutes!
